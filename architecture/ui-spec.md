# UI仕様書

## 共通ルール
- 初期表示は「支出入力」
- 下部タブ: 支出入力 / 月次サマリ / 管理
- 入力は最短3タップ以内を優先
- 通貨はJPYを前提とし、金額は整数

## 画面: 初回起動セットアップ案内

### 目的
- 初期設定未完了時に、予算設定とテンプレ管理への導線を提示する

### 表示条件
- `hasBudget = false` または `hasVisitedTemplateManagement = false` の場合に表示
- 条件を満たす間は、`MainScreen` 表示時に再表示
- ただし `hideInitialSetupAnnouncement = true` の場合は表示しない

### 画面構成
- タイトル: 「セットアップ」
- 説明文: 「最初に予算を設定しておくと、月の管理がしやすくなります。テンプレを使うと入力も早くなります。」
- アクション1: 「予算設定へ」
- アクション2: 「テンプレ管理へ」
- 閉じる操作: 「あとで」
- 補助操作: 「今後表示しない」

### 挙動
- 「予算設定へ」で予算設定画面に遷移する
- 「テンプレ管理へ」でテンプレ管理画面に遷移する
- 「あとで」でダイアログを閉じる（条件未達の場合、次回 `MainScreen` 表示時に再表示）
- 「今後表示しない」で `hideInitialSetupAnnouncement = true` を保存し、次回以降は表示しない
- 予算設定済み かつ テンプレ管理訪問済みになったら表示しない
- 設定画面の「初期設定ガイドを再表示」で `hideInitialSetupAnnouncement = false` に戻せる

### メッセージ
- 画面内文言はシンプルな短文を使用する

## 画面: バックアップ

### 目的
- Google Drive AppDataへデータをExport/Importする

### 画面構成
- Googleサインインボタン
- 接続中アカウント表示
- Exportボタン
- Import（全置換）ボタン

### 挙動
- Export時は `backupSchemaVersion` を含むJSONを生成してDrive AppDataに保存
- Import時は `backupSchemaVersion` に応じたマイグレーションを適用して復元
- Importは全置換で実行し、`EXCLUDED` データは復元後に再構築する

### エラーメッセージ
- Googleアカウントでサインインしてください
- バックアップファイルが見つかりません
- このバックアップバージョンには未対応です

### メッセージ
- Export成功時に「Google Driveにバックアップしました」を表示
- Import成功時に「Google Driveから復元しました」を表示

## 画面: 月次サマリ

### 目的
- 月の予算と支出の状況を把握する

### 画面構成
- 月選択（当月がデフォルト、過去月へ遡れる）
- 今月の残り予算カード（当月予算 - 当月支出 + 臨時収入）
- 予算 / 支出のサマリ
- 臨時収入の別枠合計（残り予算計算に含まれている）
- 貯金目標進捗カード（モード表示 / 繰越金 / 達成状況）
- カテゴリ別支出グラフ
- カテゴリ別まとめ（カテゴリごとの合計一覧、全件表示）
- 支出一覧（当月全件スクロール、月日表示、新しい順）
- ナビゲーション: 下部タブを使用

### 再設計方針（モノクロ）
- 強調は色ではなく「サイズ・太字・余白・角丸・線幅」で行う
- 情報の優先順位: 今月の残り予算 > 予算/支出 > 貯金目標 > カテゴリ/一覧

### レイアウト構成（上から順）
- 月ヘッダー（YYYY年M月 + 前月/翌月矢印 + 今月に戻る）
- 今月の残り予算カード（残り予算ラベル + 大きい数字 + 予算/支出 + 進捗バー）
- 貯金目標カード（モードラベル + 繰越金 + 目標カード一覧または合計達成サマリ）
- カテゴリ別支出（グラフ + Top3 + 一覧リンク）
- 最近の支出（最新5件 + 全件表示リンク）

### タイポ/余白
- 残り予算数字: 34sp, Bold
- 残り予算ラベル: 14sp, Medium
- 予算/支出: 14sp, Regular
- セクションタイトル: 16sp, SemiBold
- 一覧行: 15sp, Regular
- リンク文言: 12sp, Medium
- 画面左右パディング: 16dp
- セクション間の縦余白: 16dp
- 今月の残り予算カード内パディング: 16dp
- 進捗バー高さ: 6dp
- 角丸: 今月の残り予算カード20dp、通常カード12dp

### 月選択の挙動
- 左右スワイプで月移動（メイン導線）
- 矢印タップでも月移動（補助導線）
- 「今月に戻る」は当月以外のときだけ表示
- 当月は「今月」ラベルを小さく併記
- 空月は予算/支出/残額を0表示

### カテゴリ別グラフ
- Top3のみ表示（その他は「その他」へ集約）
- 表示順は金額の降順
- 0件のときは「データがありません」
- 「一覧を見る ＞」でカテゴリ別一覧へ遷移

### 支出一覧
- 最新5件まで表示、表示順は日付降順
- 未分類は「未分類」固定表示
- 0件のときは「今月の支出はありません」
- 「全件表示 ＞」で支出一覧へ遷移
- 行タップで編集、スワイプで削除（削除前確認ダイアログ）

### インタラクション
- 画面入場: 今月の残り予算カード/進捗バーを先にフェードイン（150ms）
- 貯金目標カードは今月の残り予算カードの直後に表示し、要約値を先に認識できるようにする
- グラフ→一覧はスタッガー表示（各80ms）
- 月切替: コンテンツ全体スライド（120ms）
- 数値変化: クロスフェード（150ms）

### 貯金目標カードの表示ルール
- 先頭に現在モードを示すラベルを表示する
  - 個別達成モード: 「個別達成中」
  - 合計達成モード: 「合計達成中」
- 要約領域に「繰越金」を表示する
- 個別達成モードでは目標ごとにカードを表示し、目標名・目標金額・状態（達成 / あと○円）を表示する
- 合計達成モードでは「合計目標」「達成状況」を強調表示する
- 目標未登録時は「貯金目標が未登録です」を表示し、詳細カードは表示しない

### 挙動
- 月切替で表示内容を更新
- 月選択で過去月へ遡れる

### エラーメッセージ
- データ取得に失敗しました

### 操作例
- 「今月」をタップして当月に戻る
- 未分類の支出にカテゴリを付けて保存

### メッセージ
- 編集完了時に「支出を更新しました」トースト表示
- 削除前に「この支出を削除しますか？」確認ダイアログ

## 画面: カテゴリ管理

### 目的
- カテゴリとサブカテゴリの管理

### 画面構成
- カテゴリ一覧（階層表示）
  - 各カテゴリをタップするとサブカテゴリ一覧を展開
  - カテゴリごとに追加 / 編集 / 削除
  - ドラッグ&ドロップで並び替え可能
- サブカテゴリはカテゴリ配下に表示
  - サブカテゴリごとに編集 / 削除
  - ドラッグ&ドロップで並び替え可能

### 並び替えの挙動
- 親カテゴリ同士の並び替えが可能
- サブカテゴリは親カテゴリ内で並び替え可能
- ドラッグ開始時にアイテムを視覚的に強調表示
- ドロップ時に新しい並び順を自動保存
- 並び順が未設定のカテゴリは作成日時順で表示

### バリデーション
- カテゴリ名は必須
- サブカテゴリ名は必須
- カテゴリは最大10件まで
- サブカテゴリは各カテゴリごとに最大10件まで

### エラーメッセージ
- カテゴリは最大10件です
- サブカテゴリは最大10件です
- カテゴリ名を入力してください
- サブカテゴリ名を入力してください
- 保存に失敗しました
- 並び順の保存に失敗しました

### 操作例
- カテゴリ「趣味」を追加
- 「趣味」をタップしてサブカテゴリ一覧を表示
- サブカテゴリ「本」を追加
- 「本」を削除
- カテゴリをドラッグ&ドロップで並び替え
- サブカテゴリをドラッグ&ドロップで並び替え

### メッセージ
- 保存完了時に「カテゴリを保存しました」トースト表示
- 削除前に「このカテゴリを削除しますか？」確認ダイアログ
- 並び替え完了時に「並び順を保存しました」トースト表示
